Drone Commander

Drone Commander는 Blockly 시각적 프로그래밍으로 드론 비행을 제어하는 브라우저 애플리케이션입니다. 블록을 조립하고 프로그램을 실행하면 드론이 Three.js 실시간 3D 장면에서 명령을 수행합니다.

빠른 시작: 시작 블록을 놓고 그 아래에 이륙과 착륙 블록을 연결한 다음 녹색 ▶ 버튼을 누르세요.

인터페이스 개요

크기를 조절할 수 있는 구분선을 기준으로 화면이 두 패널로 나뉩니다.

  • 왼쪽 패널: 비행 프로그램을 만드는 Blockly 편집기와 언어, 파일, 시나리오, 그래픽 제어 메뉴입니다.
  • 오른쪽 패널: 3D 드론 화면이며, 바로 아래에 비행 상태와 프로그램 변수가 표시됩니다.

오른쪽 도구 모음에는 전체 화면, 현재 파일 이름, ▶ 실행 및 ■ 정지 버튼이 있습니다. 전체 화면에서는 편집기와 상태 패널이 잠시 숨겨집니다.

Blockly 편집기

  1. 도구 상자에서 시작 블록을 끌어옵니다.
  2. 그 아래에 드론, 논리, 반복 및 기타 블록을 연결합니다.
  3. 숫자, 텍스트, 센서 또는 변수 블록을 알맞은 입력에 끼웁니다.
  4. ▶ 실행을 누릅니다. 프로그램 흐름에 연결된 블록만 순서대로 실행됩니다.
논리 반복 수학 텍스트 목록 변수 함수 센서 드론 흐름

마우스 휠로 작업 공간을 확대하고 빈 공간을 드래그해 이동할 수 있습니다. 블록을 마우스 오른쪽 버튼으로 누르면 복제, 삭제, 주석 기능을 사용할 수 있습니다. 프로그램과 파일 이름은 브라우저 로컬 저장소에 자동 저장됩니다.

드론 블록

별도의 설명이 없는 이동 명령은 드론이 이륙한 상태에서만 동작합니다. 거리와 고도는 시뮬레이터 단위를 사용합니다.

블록설명
이륙프로펠러를 시작하고 기본 비행 고도 10까지 상승합니다.
착륙현재 위치의 지면 높이까지 내려가 프로펠러를 멈춥니다.
고도 설정 [N]지정한 절대 고도로 부드럽게 이동합니다.
고도 변경 [N]상대값만큼 상승하거나 하강하며 안전 고도 아래로 내려가지 않습니다.
각도 설정 [N]방향을 0~359도의 절대 각도로 설정합니다.
각도 변경 [N]상대 각도만큼 회전합니다. 예: 90은 오른쪽, -90은 왼쪽 회전입니다.
옆으로 이동 [N]양수는 오른쪽, 음수는 왼쪽으로 이동합니다.
전진 [N]드론이 바라보는 방향으로 지정한 거리만큼 이동합니다.
전진 [N] 상승 [N]앞으로 이동하면서 고도를 상대적으로 변경합니다.
이동 위치 x y z절대 좌표로 이동하며 Y는 고도를 뜻합니다.
상대 이동 x y z현재 방향 기준으로 X는 오른쪽/왼쪽, Y는 위/아래, Z는 앞/뒤입니다.
절대 곡선현재 위치에서 절대 중간점을 지나 절대 목적지까지 3D 원호로 비행합니다.
상대 곡선현재 방향을 기준으로 한 중간점과 목적지 오프셋을 사용해 3D 곡선으로 비행합니다.
기지로 복귀X=0, Z=0, 고도 10인 기지로 돌아갑니다.
대기 [N]지정한 초만큼 프로그램을 일시 정지합니다. 이륙 상태가 아니어도 됩니다.
연기 [참/거짓]비행 경로를 보여 주는 연기 효과를 켜거나 끕니다.
속도 [N]이동 속도를 0~10으로 설정합니다. 기본값은 1입니다.
좌표 참고: 상대 이동과 상대 곡선은 드론 방향에 따라 회전합니다. Z는 항상 앞/뒤, X는 오른쪽/왼쪽입니다. 절대 좌표는 회전하지 않습니다.

센서와 흐름

블록반환값설명
키 [KEY] 눌림불리언방향키, 스페이스, 엔터 또는 A~Z를 감지합니다. 조건과 반복문에서 사용할 수 있습니다.
X / Z숫자드론의 현재 수평 좌표를 반환합니다.
고도숫자현재 고도를 반환합니다.
방향숫자현재 방향 각도를 반환합니다.
속도숫자현재 속도 설정을 반환합니다.
시작흐름프로그램의 시작점입니다. 명령을 이 블록 아래에 연결합니다.
끝흐름선택적으로 사용하는 프로그램 종료 표시입니다.

실행 및 카메라 제어

  • ▶ 실행: 장면을 초기화한 뒤 Blockly 프로그램을 단계별로 실행합니다. 실행 중에는 실행 버튼이 비활성화됩니다.
  • ■ 정지: 프로그램과 현재 동작을 즉시 멈춥니다. 다시 누르면 드론을 초기화합니다.
  • 오른쪽 버튼 드래그: 드론 주위로 카메라를 회전합니다.
  • 마우스 휠: 확대하거나 축소합니다.
  • 가운데 버튼: 드론 뒤쪽 시점으로 돌아갑니다.
  • 왼쪽 버튼: 기본 카메라로 복원하며, 비행 중에는 추적 시점으로 전환할 수 있습니다.

“성능”과 “균형” 그래픽에서는 지면 표시와 파란 고도선으로 드론 위치와 높이를 확인할 수 있습니다.

상태와 변수

3D 화면 바로 아래 패널에는 다음 정보가 실시간으로 표시됩니다.

  • X, Z, 고도, 방향: 편집할 수 있습니다. 값을 바꾸고 Enter를 누르면 지형 충돌을 확인한 후 위치나 방향이 즉시 변경됩니다.
  • 상태: 드론이 “지상”인지 “비행 중”인지 표시합니다.
  • 변수: 프로그램 실행 중 모든 Blockly 변수와 현재 값을 한 줄에 하나씩 표시합니다. 초기화 전에는 “—”가 보이며 실행 종료 후 마지막 값이 유지됩니다.

파일, 시나리오 및 그래픽

  • 새로 만들기: 시작 블록만 있는 새 프로그램을 만듭니다.
  • 저장: 프로그램을 Blockly XML 파일로 다운로드합니다.
  • 불러오기: XML 파일로 현재 프로그램을 교체합니다.
  • 자동 저장: 변경할 때마다 현재 브라우저에 저장됩니다. 로컬 저장소를 지우면 데이터도 삭제됩니다.

시나리오 메뉴에서 비행장, 도심 트랙, 대도시, 열대 섬을 선택할 수 있습니다. 시나리오를 바꾸면 드론과 카메라가 초기화됩니다.

그래픽용도
성능낮은 해상도와 비활성화된 그림자로 저사양 장치에 적합합니다.
균형화질과 속도를 조절한 기본 설정입니다.
고품질높은 해상도, 그림자, 자세한 연기를 사용하며 성능이 좋은 장치에 적합합니다.

언어, 시나리오, 그래픽 설정과 파일 이름은 브라우저에 저장됩니다. 언어 메뉴는 English, Italiano, Français, Deutsch, Español, Português, العربية, 简体中文, 한국어, 日本語를 지원합니다.

사용 팁과 문제 해결

  • 항상 시작으로 프로그램을 시작하고 이동 명령 전에 이륙을 사용하세요.
  • 연기로 경로를 확인하고 속도로 이동 속도를 조절하세요.
  • 드론은 지면과 장면의 장애물을 통과하지 않으며 고도는 안전한 값으로 자동 제한됩니다.
  • 드론이 움직이지 않으면 상태가 “비행 중”인지, 블록이 프로그램 흐름에 연결되어 있는지 확인하세요.
  • 3D 화면이 비어 있으면 WebGL을 지원하는 최신 Chrome, Firefox 또는 Edge를 사용하세요.
  • 실행이 느리면 “성능” 그래픽을 선택하세요.
  • 소리는 브라우저 자동 재생 정책 때문에 사용자가 ▶ 버튼을 누른 뒤 시작됩니다.
